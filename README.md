# Blender_4_Rhino

DESCRIPTION

"A Blender 4 Rhino" in short is a mesh Blender for Rhino (bridge).
Let you export form Rhinoceros 3D with one (custom) botton and import that mesh quicklly inside Blender (using this plugin).
